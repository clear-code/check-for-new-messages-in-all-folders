# Check New Messages in All Folders 試験項目

## 準備

以下のメールアカウントを用意する。

 * ローカルフォルダアカウント（最初から存在する物）
 * POP3メールアカウント（名前は任意）
 * IMAPメールアカウント1（名前は任意）
 * IMAPメールアカウント2（名前は任意）

また、検証用IMAPメールアカウント1について、以下の用意をする。

 * 以下の構成で入れ子になったフォルダを作成しておく。
   - テスト1
     - テスト2
     - テスト3
 * 各フォルダの「このフォルダに新着メッセージがないか確認する」にチェックが入っていないことを確認する。

## 試験項目

### メニューバーのメニューの初期化の試験

 1. メニューバーから「ツール」→「新着メールの確認」と辿る。
 2. IMAPメールアカウント1と2の名前がメニュー項目として表示されていることを確認する。
 3. IMAPメールアカウント1のサブメニューを辿る。
 4. 以下の2つのメニュー項目があることを確認する。
    * すべてのフォルダの新着メールを確認する（チェックなし）
    * 受信トレイの新着メールのみ確認する（チェックあり）

### アプリケーションメニューのメニューの初期化の試験

 1. アプリケーションメニューから「ツール」→「新着メールの確認」と辿る。
 2. IMAPメールアカウント1と2の名前がメニュー項目として表示されていることを確認する。
 3. IMAPメールアカウント1のサブメニューを辿る。
 4. 以下の2つのメニュー項目があることを確認する。
    * すべてのフォルダの新着メールを確認する（チェックなし）
    * 受信トレイの新着メールのみ確認する（チェックあり）

### チェック状態の混在時の挙動の試験

 1. IMAPメールアカウント1のフォルダのうち1つだけについて、プロパティを開き「このフォルダに新着メッセージがないか確認する」のチェックをONにする。
 2. メニューバーから「ツール」→「新着メールの確認」→IMAPメールアカウント1のメニュー と辿る。
 3. 「受信トレイの新着メールのみ確認する」にチェックが入った状態であることを確認する。

### すべてのフォルダのチェック状態をONに変更する試験

 1. メニューバーから「ツール」→「新着メールの確認」→IMAPメールアカウント1のメニュー と辿る。
 2.  「すべてのフォルダの新着メールを確認する」を選択する。
 3. IMAPメールアカウント1の受信トレイ以外の各フォルダのプロパティを開き、すべてのフォルダについて「このフォルダに新着メッセージがないか確認する」のチェックがONになっていることを確認する。
 4. メニューバーから「ツール」→「新着メールの確認」→IMAPメールアカウント1のメニュー と辿る。
 5.  「すべてのフォルダの新着メールを確認する」にチェックが入っていることを確認する。

### すべてのフォルダのチェック状態をOFFに変更する試験

 1. メニューバーから「ツール」→「新着メールの確認」→IMAPメールアカウント1のメニュー と辿る。
 2.  「受信トレイの新着メールのみ確認する」を選択する。
 3. IMAPメールアカウント1の受信トレイ以外の各フォルダのプロパティを開き、すべてのフォルダについて「このフォルダに新着メッセージがないか確認する」のチェックがOFFになっていることを確認する。
 4. メニューバーから「ツール」→「新着メールの確認」→IMAPメールアカウント1のメニュー と辿る。
 5.  「受信トレイの新着メールのみ確認する」にチェックが入っていることを確認する。
